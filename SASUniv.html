    Information Pages on SAS University Edition.

    SAS University Edition is a FREE version of SAS that you can download, install and run for FREE. Please take a look at the following files to get you started:

    • SASUnivSetup_mkh.pdf - Setup Instructions Including Some Examples to get you up and running
    • mkh1.sas - the initial SAS program to download and use to run the examples
    • AgeOrionsData.sas7bdat - the example dataset used - download to run the example program
